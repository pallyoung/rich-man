# RichMan — 中国股市量化分析平台设计文档

## 1. 概述

RichMan 是面向个人投资者的中国A股量化分析平台，集成行情分析、技术指标、趋势识别、策略回测和财经资讯，提供一站式投资决策辅助。

## 2. 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | React 18 + Vite | SPA框架（已迁移为 TypeScript，前端源码全部 .ts/.tsx） |
| UI组件 | Ant Design 5 | 企业级UI库 |
| 图表 | ECharts 5 | K线、技术指标、热力图 |
| 后端 | Flask (Python) | REST API |
| 数据源 | AKShare | 免费A股数据 |
| 技术指标 | TA-Lib + pandas | 指标计算引擎 |
| 存储 | SQLite | 本地缓存 |
| 主题 | CSS Variables | 深色/浅色切换 |

## 3. 核心模块

### 3.1 行情中心
- 主要指数总览（上证、深证、创业板、科创50）
- 涨跌排行榜（涨幅、跌幅、换手率、成交额）
- 板块热力图（行业/概念板块涨跌分布）
- 龙虎榜数据

### 3.2 个股分析
- K线图（日/周/月/年线，支持前复权/后复权）
- 分时图（当日分时走势）
- 技术指标叠加：MA(5/10/20/60/120/250), MACD, KDJ, RSI, BOLL, VOL
- 筹码分布图
- 基本面数据（市盈率、市净率、营收、净利润等）

### 3.3 趋势分析
- 趋势识别：上升/下降/横盘判断
- 多空信号：金叉/死叉自动标注
- 买卖点提示：基于MACD+KDJ+RSI多指标共振
- 支撑/压力位计算（基于前期高低点、均线、布林带）
- 多股趋势对比

### 3.4 量化策略
- 内置策略模板：
  - 双均线策略（MA金叉死叉）
  - MACD策略
  - 海龟交易策略
  - 动量策略
- 策略回测引擎：
  - 时间范围选择
  - 初始资金设置
  - 手续费/滑点模拟
- 回测结果：
  - 收益曲线
  - 年化收益率、最大回撤、夏普比率、胜率
  - 交易明细列表
- 因子选股：
  - 基本面因子（PE、PB、ROE）
  - 技术面因子（动量、波动率）
  - 多因子综合评分

### 3.5 资讯中心
- 实时财经新闻流（抓取主流财经网站）
- 个股公告查询
- 市场情绪指数（基于新闻NLP分析）
- 热点关键词云

## 5. API设计

### 行情接口
- `GET /api/market/overview` — 市场总览（指数、涨跌家数）
- `GET /api/market/ranking?type=rise&page=1` — 涨跌排行
- `GET /api/market/sectors` — 板块数据
- `GET /api/market/hot` — 龙虎榜

### 个股接口
- `GET /api/stock/{code}/kline?period=daily` — K线数据
- `GET /api/stock/{code}/realtime` — 实时行情
- `GET /api/stock/{code}/indicators` — 技术指标
- `GET /api/stock/{code}/fundamental` — 基本面

### 趋势接口
- `GET /api/trend/signals` — 趋势信号列表
- `GET /api/trend/compare?codes=xxx,xxx` — 多股对比
- `GET /api/trend/sector-rotation` — 行业轮动

### 量化接口
- `POST /api/quant/backtest` — 策略回测
- `POST /api/quant/factor-select` — 因子选股
- `GET /api/quant/strategies` — 策略列表

### 资讯接口
- `GET /api/news/latest?page=1` — 最新新闻
- `GET /api/news/stock/{code}` — 个股新闻
- `GET /api/news/sentiment` — 市场情绪

## 6. 数据缓存策略

| 数据类型 | 缓存时间 | 说明 |
|----------|----------|------|
| 实时行情 | 5分钟 | 减少API调用 |
| K线数据 | 当日收盘后更新 | 历史数据不变 |
| 技术指标 | 随K线更新 | 计算结果缓存 |
| 财务数据 | 1天 | 季报级更新 |
| 新闻资讯 | 30分钟 | 保持时效性 |
| 板块数据 | 10分钟 | 中等频率 |

## 7. UI主题系统

使用CSS Variables实现深色/浅色主题切换：

- **深色主题**（默认）：#0d1117背景，#58a6ff强调色，类似同花顺专业版
- **浅色主题**：#ffffff背景，#1677ff强调色，类似雪球风格
- 切换方式：导航栏右上角主题开关，localStorage持久化
